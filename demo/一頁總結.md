**你會看到**：3D 軌跡圖、位置分析、速度圖、動畫影片  
**時間**：1-2 分鐘

### 2️⃣ 測試指令（不需訓練）

**範例指令**：「把芒果移到左邊」、「抓住芒果並抬高」  
**時間**：每個指令 5 秒

### 3️⃣ 訓練模型（可選）
```bash
bash quick_train_demo.sh
```
**時間**：10-30 分鐘（1000 steps）

### 4️⃣ 視覺化模型結果
```bash
python3 visualize_inference.py \
    --checkpoint ./outputs/demo_quick/last.ckpt \
    --num_episodes 5
```
**你會看到**：模型預測 vs 真實軌跡對比

## 💡 關鍵概念

### Episode（操作示範）
- 一個 episode = 一次完整的操作（從開始到結束）
- 包含：影像序列 + 手臂軌跡 + 指令
- 你有 181 個 episodes

### 模型訓練
```
輸入：[影像 + "把芒果移到左邊" + 當前位置]
  ↓ 模型學習
輸出：[接下來 64 步的動作序列]
```

### 動作向量（10維）
```python
[x, y, z,           # 位置（3維）
 r1, r2, r3,        # 旋轉（6維）
 r4, r5, r6,
 gripper]           # 夾爪（1維：0=閉，1=開）
```

## ❓ 常見問題速答

**Q: 我需要 GPU 嗎？**  
A: 視覺化不需要。訓練建議要（但 CPU 也能跑，只是慢）

**Q: 必須訓練模型嗎？**  
A: 不用！前兩個步驟就能看到效果

**Q: 訓練要多久？**  
A: 快速測試 10-30 分鐘，完整訓練數小時

**Q: 可以改指令嗎？**  
A: 可以！`interactive_demo.py` 支援任何中文指令

**Q: 如何部署到實體機器人？**  
A: 先完成視覺化測試，再用 ROS2 介面（repo 中有範例）
